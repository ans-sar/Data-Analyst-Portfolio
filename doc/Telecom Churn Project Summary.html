📊 Telecom Customer Churn Analysis 📊

🎯 Project Overview 🎯

Customer retention is a crucial factor in the telecom industry. 🎯 Objectives 🎯

  • Visualize and analyze customer churn trends across different dimensions 📊

  • Study churner profiles to help implement targeted marketing strategies 🎯

  • Predict potential future churners using defined KPIs 🤖

🛠 Step 1: Data Cleaning & Processing 🛠

  • Created a Monthly Charge Range column for grouping values:
